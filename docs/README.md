# Documentation Index

This folder contains the project documentation for running and maintaining the Depolarization workflows.

## Reading Order

1. `getting_started.md`: environment setup, first run, CLI/REPL usage.
2. `configuration.md`: config schema, overrides, task parameters.
3. `jobs.md`: per-job inputs, outputs, and execution order.
4. `instrumental_effect.md`: detailed contract for the instrumental-effect pipeline.
5. `development.md`: architecture, tests, and contribution workflow.
6. `functions.md`: auto-generated function index from `src/code/`.

## Quick Links

- Setup and first execution: `getting_started.md`
- Full task list and outputs: `jobs.md`
- Instrumental diagnostics (integrity, alignment): `instrumental_effect.md`
- REPL built-in helper: `DepolLib.depol_help()`

## Source of Truth

- Runtime behavior is defined by the Julia code in `src/code/`.
- `docs/functions.md` is generated by `scripts/generate_functions_doc.sh`.
- Configuration examples come from `config/default.toml`.
